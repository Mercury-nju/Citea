## 📝 技术细节

### CSS 平滑滚动
```css
html {
  scroll-behavior: smooth;
}
```

### JavaScript 滚动
```typescript
// 滚动到指定元素
element?.scrollIntoView({ behavior: 'smooth' })

// 滚动到顶部
window.scrollTo({ top: 0, behavior: 'smooth' })
```
